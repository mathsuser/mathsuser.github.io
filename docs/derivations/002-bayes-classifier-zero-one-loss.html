Question

Given a binary classification problem with \(Y \in \{0,1\}\), derive the Bayes classifier under the \(0\)-\(1\) loss and calculate the corresponding Bayes risk.

Setup

We recall the following definitions:

  1. The \(0\)-\(1\) loss is defined as \[ L(y, \hat{y}) = \mathbf{1}_{\{y \neq \hat{y}\}} = \begin{cases} 0, & \text{if } y = \hat{y}, \\ 1, & \text{if } y \neq \hat{y}. \end{cases} \]
  2. The risk of a classifier \(g:\mathbb{R}^d \to \{0,1\}\) under \(0\)-\(1\) loss is \[ R(g) = \mathbb{E}[L(Y, g(X))] = \mathbb{P}(Y \neq g(X)). \]

We also recall that the Bayes optimal classifier \(g^*\) is the classifier that minimises the risk over all possible classifiers. The corresponding minimum risk is called the Bayes risk \(R^*\). More formally, \[g^* = \arg\min_{g} R(g), \quad R^* = R(g^*).\]

We also recall the following lemma which provides an alternative approach to find a Bayes optimal classifier:

NoteLemma (Bayes optimal classifier)

If \(g^\ast\) is a classifier that minimises the conditional risk \[ R^{cond}(g \mid X=x)= \int L(y, g(x)) dP_{Y|X=x}(y) \] for almost every \(x\), then \(g^\ast\) is a Bayes optimal classifier.

TipProof

Let \(g\) be any classifier. Then we can write the risk of \(g\) as \[\begin{aligned} R(g) &= \mathbb{E}[L(Y, g(X))] = \mathbb{E}[L(Y, g(X))X] \\\ &= \int \int L(y, g(x)) dP_{Y|X=x}(y) dP_X(x) \\\ &= \int R^{cond}(g \mid X=x) dP_X(x). \end{aligned}\] Since \(g^\ast\) minimises the conditional risk for almost every \(x\), we have \[R^{cond}(g^\ast \mid X=x) \le R^{cond}(g \mid X=x)\] for almost every \(x\). Integrating both sides with respect to \(P_X\) gives \[R(g^\ast) = \int R^{cond}(g^\ast \mid X=x) dP_X(x) \le \int R^{cond}(g \mid X=x) dP_X(x) = R(g).\] Since \(g\) was arbitrary, we conclude that \(g^\ast\) is a Bayes optimal classifier.

The above lemma states that to find a Bayes optimal classifier, it suffices to find a classifier that minimises the conditional risk for each fixed \(x\). This is often easier than directly minimising the overall risk, as the minimisation problem involves only the conditional distribution of \(Y\) given \(X=x\) rather than the joint distribution of \((X,Y)\).

Derivation

  1. Let \(g:\mathbb{R}^d \to \{0,1\}\) be any classifier. The conditional risk of \(g\) under \(0\)-\(1\) loss can be expressed as

\[\begin{aligned} R^{cond}(g \mid X=x) &= \mathbb{P}(Y \neq g(X) \mid X=x) \\\ &= \mathbb{P}(Y=1 \mid X=x) \mathbf{1}_{g(x)=0} + \mathbb{P}(Y=0 \mid X=x) \mathbf{1}_{g(x)=1} \\\ &= \eta(x) (1-g(x)) + (1-\eta(x)) g(x), \\\ \end{aligned}\] where \(\eta(x) := \mathbb{P}(Y=1 \mid X=x)\) is the regression function.

  1. To find the Bayes optimal classifier, it suffices to minimise the conditional risk \(R^{cond}(g \mid X=x)\) with respect to \(g(x)\) for each fixed \(x\). Since \(g(x)\) can only take values in \(\{0,1\}\), we can evaluate the conditional risk for both cases:
  • If \(g(x)=0\), then \[R^{cond}(g \mid X=x) = \eta(x).\]
  • If \(g(x)=1\), then \[R^{cond}(g \mid X=x) = 1-\eta(x).\]

  1. The conditional risk is minimal if we choose \(g(x)=1\) when \(\eta(x) > 1-\eta(x)\), and \(g(x)=0\) otherwise. This can be rewritten as \[g^*(x) = \mathbf{1}_{\{\eta(x) > 1/2\}}.\]

  2. By the above lemma, \(g^*\) is a Bayes optimal classifier. Hence, the Bayes optimal classifier under \(0\)-\(1\) loss is given by \[g^*(x) = \mathbf{1}_{\{\eta(x) > 1/2\}}.\]

  3. The Bayes risk can be calculated as \[\begin{aligned} R^* &= R(g^*) = \mathbb{E}[L(Y, g^*(X))] = \mathbb{P}(Y \neq g^*(X)) \\\ &= \mathbb{E}[\min\{\eta(X), 1-\eta(X)\}]. \end{aligned}\]

Remarks:

  1. In the context of statistical learning theory, the variable \(X\) is often referred to as the feature or input, while the variable \(Y\) is referred to as the label or output. The regression function \(\eta(x)\) can be interpreted as the probability that the label is \(1\) given the feature \(x\). The conditional random variable \(Y\mid X\) is distributed according to a Bernoulli distribution with parameter \(\eta(X)\), which means that \(Y\) takes the value 1 with probability \(\eta(X)\) and the value 0 with probability \(1-\eta(X)\). None of the distributions of \(X\) or \(Y\mid X\) are assumed to be known in practice.

  2. Intuitively, the Bayes classifier knows the regression function \(\eta\). It is the classifier we would use if we had access to the true distribution of \(Y\mid X\).

  3. In practice, we cannot predict \(Y\) accurately. The classification error associated with a chosen classifier is not zero. The Bayes risk provides a lower bound on the classification error that can be achieved by any classifier. It is the best possible error rate that can be achieved given the distribution of the data and the noise in the labels.

  4. While optimal, the Bayes classifier presents a major drawback: we cannot compute it because we do not know the regression function \(\eta(x)\). One of the main goals of statistical learning is to estimate \(\eta(x)\) from data, and then use the estimated regression function to construct a classifier that approximates the Bayes optimal classifier. For instance, one can assume a parametric model for \(\eta(x)\), such as logistic regression.
